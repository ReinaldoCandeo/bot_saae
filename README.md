# 🏪 Loja do Japão - Site E-commerce

## 🎯 Visão Geral
Site e-commerce moderno com tema japonês, desenvolvido com HTML, CSS e JavaScript puro. Inspirado em designs modernos de e-commerce com foco em produtos japoneses autênticos.

## 🚀 Características

### **Design e Tema**
- ✅ **Nome:** "LOJA DO JAPÃO" em vermelho
- ✅ **Cores:** Vermelho (#dc2626) como cor principal
- ✅ **Tema:** Produtos japoneses autênticos
- ✅ **Layout:** Design moderno e responsivo

### **Funcionalidades**
- 🔍 **Sistema de busca** funcional
- 🛒 **Carrinho de compras** com contador
- 📱 **Design responsivo** para mobile e desktop
- 💬 **Botão WhatsApp** flutuante
- 🎨 **Animações** e efeitos hover
- 🔔 **Sistema de notificações**

## 📁 Estrutura do Projeto

```
japaloja/
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # Funcionalidades JavaScript
└── README.md           # Documentação
```

## 🎨 Seções do Site

### **1. Header (Cabeçalho)**
- Logo "LOJA DO JAPÃO" em vermelho
- Campo de busca centralizado
- Links de login/cadastro
- Ícone do carrinho com contador

### **2. Navegação**
- Menu "Departamentos"
- Links para categorias (Eletrônicos, Telefonia, etc.)
- Botões especiais (Blog, Promoções)

### **3. Banner Principal**
- Título: "Lançamento iPhone 17 Series"
- 3 produtos em destaque com fotos reais
- Botão "Conhecer todos"
- Seta de navegação

### **4. Categorias**
- 5 categorias com imagens reais:
  - Bebidas (Sake e Chás)
  - Eletrônicos (Tecnologia Japonesa)
  - Informática (Gadgets Premium)
  - Telefonia (Smartphones)
  - Som & Áudio (Equipamentos Hi-Fi)

### **5. Produtos em Destaque**
- Grid de 8 produtos com fotos reais
- Preços em destaque
- Botões "Adicionar ao Carrinho"
- Badges de "Novo" e "Promoção"

### **6. Footer**
- Informações da loja
- Links úteis
- Redes sociais
